# umme-ToDo-Command-Line-To-Do-List-Manager-C-
A simple, clean, and functional command-line To-Do List App written in C++. Built using Object-Oriented Programming, STL lists, iterators, and a minimal text-based UI.  This project was created to practice:  OOP (classes, objects, encapsulation)  STL containers (std::list)  Iterators  CLI-based application design  Basic CRUD operations
